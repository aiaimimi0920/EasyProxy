# Milestones

## Milestone 1: Planning Baseline

Deliverables:

- architecture docs
- migration plan
- analysis and risk notes

Exit condition:

- target structure and copy rules are documented

## Milestone 2: First Structural Import

Deliverables:

- copied `service/`, `upstreams/`, `workers/`, and `deploy/` trees
- repeatable sync script

Exit condition:

- all source modules exist in their target locations

## Milestone 3: Monorepo Readability

Deliverables:

- root README
- contributing guide
- quickstart and upstream sync docs

Exit condition:

- an external contributor can discover the repository layout from the root

## Milestone 4: Follow-Up Cleanup

Deliverables:

- path cleanup inside imported docs
- root CI strategy
- optional shared config strategy

Exit condition:

- the monorepo can evolve independently from the legacy workspace layout
